Sopro Pulse adds the tracking code from Sopro campaigns.

== Description ==

Sopro Pulse adds the tracking code from Sopro campaigns. Verification is done within the Portal.


== Installation ==

1. Upload `sopro-pulse.zip` with WordPress plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the tracking code supplied by our team
